# Jan 2026 DLGenAI Project - Messy Mashup  
**Predicting Music Genre from Noisy Mashups**

This project is part of the **Jan 2026 Deep Learning & Generative AI (DLGenAI) Project**.

The goal is to **predict the music genre** of a noisy audio mashup.  
Each mashup is created by mixing instrument stems (drums, vocals, bass, others) from different songs of the same genre, with tempo changes and added noise.

The task focuses on building models that are **robust to noise and variations**, similar to real-world music classification problems.

---

## 🎯 Task

Given a noisy audio mashup, predict one of the following genres:

```

blues, classical, country, disco, hiphop,
jazz, metal, pop, reggae, rock

```
---

Submissions are evaluated using **Macro F1 Score**.

This repository is in the **initial setup stage**.  
Code, experiments, and results will be added gradually as the project progresses.
